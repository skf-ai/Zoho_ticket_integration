# Logging in to the LMS

The learning portal is at https://lms.siddhantaknowledge.org/. Students sign in with the username and
password sent to their registered email address when they were enrolled.

## Cannot log in / password not working

1. Check the username is exactly as it appears in the welcome email. It is
   usually the registered email address, not a nickname or roll number.
2. Passwords are case-sensitive. Ask the student to check Caps Lock, and to type
   the password rather than paste it (pasting often carries a trailing space).
3. Use the **Forgot password** link on the login page. A reset link goes to the
   registered email address and is valid for a limited time.
4. If the reset email does not arrive, check the spam and promotions folders and
   search the inbox for "https://lms.siddhantaknowledge.org/" or "password".
5. Try a different browser, or a private/incognito window, and clear the cache.

If the student has done all of the above and still cannot log in, this needs the
LMS admin — raise a ticket. Common causes that only an admin can fix: the account
was never activated, the email address on file is wrong, or the account is
suspended.

## "Invalid login, please try again" repeatedly

Moodle temporarily locks an account after several failed attempts. The lockout
clears on its own after a short wait. If the student is certain the password is
correct and it still fails after waiting, raise a ticket — the password may need
an admin reset.

## Account locked

Repeated failed attempts lock the account. Waiting usually clears it. If the
student is blocked and cannot wait (for example during an exam), raise a ticket
and mark it urgent in the description.
